# Core Features

## 1. Real-Time Tracking
Integration with Leaflet maps and Socket.IO allows both users and drivers to see each other's live positions on an interactive map.

## 2. Secure OTP Verification
To ensure passenger safety and prevent unauthorized ride starts, a 4-digit OTP is generated upon booking and must be verified by the driver.

## 3. Dynamic Fare Calculation
Calculates price based on Haversine distance formulas, vehicle type base rates, and time estimates.

## 4. Modern Glassmorphism UI
A premium dark-themed interface built with custom CSS variables, translucent panels, and responsive grid layouts.

## 5. Role-Based Dashboards
Completely separate UI experiences and navigation menus depending on whether the logged-in user is a Passenger or a Driver.
